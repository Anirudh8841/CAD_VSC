◆ getOutputFormat()

int getOutputFormat ( )

This function gets the information on what format does the user wants its output. It takes no parameters

Returns
int corresponding to the format selected by the user amongst the checklist of options

Options in the checklist would be -Save to a file: In this case user will be asked for the output file name preferred. -Isometric view: Isometric view of the user's object will be rendered on the screen irrespective of whether user has input an object of class 2DObject or 3DObject. -Orthographic view: Application will draw orthographic view of the user's object in case the input object was of class 3DObject
